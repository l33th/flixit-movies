## Usage PNPM

```bash
git clone https://github.com/l33th/flixit-movies-series-project
cd flixit-movies-series-project
pnpm install
pnpm run dev
```

## Usage NPM

```bash
git clone https://github.com/l33th/flixit-movies-series-project
cd flixit-movies-series-project
npm install
npm run dev
```

> When the server is up and running, open your browser and navigate to localhost:3000
